# Feature Flag Snapshots

This directory is generated by `scripts/snapshot_feature_flags.py`.

Artifacts:

- `lifecycle.json`: machine-readable snapshot of `codex features list`, docs coverage gaps, and websocket flag semantics derived from `openai/codex` source.
- `lifecycle.md`: human-readable report for quick review.

Do not hand-edit `lifecycle.json` or `lifecycle.md`; regenerate them with:

```bash
python3 scripts/snapshot_feature_flags.py
```
